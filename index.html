Shiny - A simple angular plugin inspired by the logo on Google chrome site.

(For now only Chrome and Safari are supported.)

Getting started

Download Shiny from here or install with Bower:

bower install angular-shiny

Inject Shiny into your Angular app:

angular.module("someApp", ["shiny"]);

Ready to shine
